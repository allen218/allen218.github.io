1.准备工具

  1. BvSshServer(搭建ssh服务器),下载window版本
  2. git客户端

2.搭建BvSshServer

  1. 安装，选择PersonEdition（个人版本），直接选是，安装成功即可.

3.BvSshServer的配置部分(重要)

使用BvSshServer最重要的就是配置部分了，接下来我们重点讲下配置部分。BvSshServer的配置主要都在setting > open easy settings里。我们先打开open easy setting.

1.配置端口和防火墙

关于端口号，我们使用ssh默认的22;然后防火墙的设置的话，选择对所有电脑开放访问(当然，这里也可以根据自己的需要去设置)

2.用户的配置

BVSshServer关于用户的设置，有两种设置方式，一种是直接使用window本身的用户来进行访问;另一种是设置虚拟用户来进行访问。

window帐号的设置和虚拟帐号的设置是一样的，这里只讲虚拟帐号的相关设置。

关于用户的配置都在上图有了说明，但是这里需要强调一下比较容易出问题的部分。

  1. shell access type选项,一定要选择git access only，不然，关于git的相关配置不会显示出来。
  2. git type这个选项，要选择msysGit or compatible，不然，后面提交时，会出现找不到repository的问题。
  3. git binaries directory,这里需要选择git客户端安装目录下的mingw64目录下的bin目录，而不是git安装目录下的bin目录。

配置完成后，我们需要开启server，点击start server即可。

服务器启动后，接下来就是初始化仓库及访问仓库了。

4.git服务器访问

1.创建

git服务器仓库是根据用户来进行设置的，每个用户可以指定相应的仓库。

上面我们设置了G:\GitServer\repository作用admin用户的仓库位置。我们先在repository下新建一个目录用来存放项目。这里我们创建一个test.git的目录。接下来我们就初始化一个git的仓库。在test.git目录下点击 右键调出git bash here,然后使用命令：

git init --bare

这样在test.git目录下就会生成一个git仓库。

2.访问

仓库创建完成后，此时就需要客户端来进行访问了。这里主要是说下访问的地址,一般使用ssh://用户名/项目目录，如:ssh://admin@192.168.1.1/test.git来进行访问就可以了。
